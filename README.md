# Words是在QPython3上面使用的手机中英文翻译工具

### 注意：请输入 *h 或 *help 获取部分帮助


### 0.支持中英文互译

### 1.支持英文离线翻译

### 2.支持单词本保存

### 3.支持单词分类存储

### 4.支持单词的导入和导出

### 5.支持单词备份

### 6.支持自动播放单词




# 依赖第三方库

### 0.requests

### 1.Qpython自带的sl4a

### 2.如果要支持其它平台，最多只需要修改compatible.py文件，如用pygame取代sl4a
